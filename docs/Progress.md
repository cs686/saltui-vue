## Props

属性 | 类型 | 默认值| 描述
---- | ---- | ----- | ----
| percent | number | 0 | 百分比|
| status| string | normal | 状态，默认normal，有normal、exception、success四种状态|
| showInfo | bool | true | 是否显示进度数值和状态图标|
| strokeWidth | number | 8 | 进度条线的宽度，单位是px |
